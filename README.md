# Uzman Klinik Psikolog Serhat TANRIVERDİ Web Sitesi

Bu paket; ana sayfa, hakkımda, hizmetler ve iletişim/randevu formu bölümlerini içeren mobil uyumlu statik web sitesi taslağıdır.

## Dosyalar

- index.html: Sayfa içeriği
- style.css: Tasarım/stil dosyası
- script.js: Mobil menü davranışı

## Yerelde Açma

1. Klasörü açın.
2. index.html dosyasına çift tıklayın.
3. Site tarayıcıda açılır.

## Yayına Alma Seçenekleri

En kolay seçenekler:
- Netlify
- Vercel
- GitHub Pages
- Klasik cPanel hosting

## Formun Çalışması

Şu an formun action alanı geçici olarak ayarlı:

https://formspree.io/f/REPLACE_WITH_YOUR_FORM_ID

Formu aktif etmek için:
1. Formspree üzerinden ücretsiz hesap açılır.
2. Yeni form oluşturulur.
3. Verilen form adresi index.html içindeki action alanına yazılır.
4. Form gönderimleri belirlenen e-posta adresine düşer.

Alternatif olarak Netlify Forms veya özel backend de kullanılabilir.

## İçerik Güncelleme

Hakkımda ve Hizmetler bölümleri şu an taslak metinlerle doludur. Bu metinler daha sonra gerçek CV, uzmanlık alanları ve çalışma yaklaşımına göre düzenlenmelidir.
